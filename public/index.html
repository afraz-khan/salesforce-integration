Salesforce Server-to-Server Integration Sample

Using OAuth2.0 JWT Bearer Flow

code@ github

This sample app has a salesforce Connected App (supporting OAuth JWT Flow) behind the scenes which talks with salesforce rest api to access a sandbox salesforce org data.

Sandbox org has ProductItems (Inventories) in salesforce for some dummy products. 2 queries are given in the code just to represent the integration flow.

You can extend the code for your usecase.
